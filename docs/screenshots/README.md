# Screenshots Directory

This directory contains screenshots of the AI Chatbot application UI.

## Required Screenshots

Please add the following screenshots to this directory:

1. **homepage.png** - Landing page with hero section and features
2. **chat-interface.png** - Main chat interface showing the conversation area
3. **login.png** - User login page
4. **chat-session.png** - Active chat session with messages

## Screenshot Guidelines

- Format: PNG or JPG
- Resolution: Minimum 1920x1080 for desktop screenshots
- File size: Keep under 2MB per image
- Naming: Use lowercase with hyphens (e.g., `homepage.png`)

## How to Take Screenshots

1. Start the application using `docker-compose up`
2. Navigate to http://localhost:3000
3. Take screenshots of each page/interface
4. Save them in this directory with the specified names
